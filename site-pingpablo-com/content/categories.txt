# CATEGORIES REFERENCE

## Purpose
Manual tracking of Hugo category taxonomies used across blog posts.
Maintains consistency and prevents typos when adding categories to post frontmatter.

## Usage
1. Check this list before adding categories to new posts
2. Add new categories here when creating them
3. Keep sorted: sort -o categories.txt categories.txt
4. Use exact spelling in post frontmatter (case-sensitive)
